# 🩺 AI‑Prevent: Clinical Diabetes Risk Prediction & Preventive Healthcare Intelligence System

An industry‑style **Healthcare AI Clinical Decision Support System (CDSS)** that predicts diabetes risk, explains contributing health factors, and provides preventive clinical recommendations.

---

## 🚀 Project Highlights

• Built hybrid healthcare AI (**Machine Learning + Clinical Rules**)
• Developed explainable diabetes risk interpretation
• Implemented preventive recommendation engine
• Designed patient health scoring system (0–100)
• Created clinical style screening report generator
• Built interactive healthcare analytics dashboard

---

## 📌 Problem

Most diabetes ML projects only output prediction probabilities without:

• Risk explanation
• Clinical interpretation
• Preventive guidance
• Patient‑specific insights
• Healthcare decision support

This project addresses these gaps by building a **clinical‑style AI screening workflow** instead of a basic prediction model.

---

## 💡 Solution

Designed a hybrid healthcare AI architecture combining:

• ML Risk Prediction
• Rule‑Based Clinical Reasoning
• Explainable AI
• Preventive Recommendation Engine
• Patient Health Analytics
• Clinical Reporting

System outputs:

• Diabetes risk probability
• Key risk drivers
• Preventive actions
• Recommended screenings
• Health score

---

## 🏗 System Architecture

**Prediction Layer** → Random Forest diabetes prediction
**Feature Engineering** → BMI groups, age groups, lifestyle features
**Hybrid Reasoning** → ML + rule scoring
**Explainability Layer** → Personalized risk factors
**Clinical Intelligence** → Preventive recommendations
**Analytics Layer** → Health score & severity index
**Product Layer** → Streamlit dashboard

---

## ⭐ Core Features

**AI Prediction Engine**
Diabetes probability prediction using Logistic Regression, Random Forest, and XGBoost.

**Explainable AI**
Identifies top contributing risk factors.

**Clinical Recommendation Engine**
Suggests:

• Exercise improvements
• Diet improvements
• Lifestyle corrections
• Screening tests
• Medical consultation

**Patient Health Score**
Generates overall wellness score.

**Clinical Report Generator**
Produces screening style health reports.

**Healthcare Dashboard**
Interactive visualization of patient risk analytics.
